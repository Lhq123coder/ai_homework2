水印工具使用说明
================

📁 文件说明：
- 水印工具.py          # 主程序文件（推荐使用）
- 运行水印工具.bat     # Windows批处理启动文件
- watermark_standalone.py  # 备用版本

🚀 快速开始：

方法1：直接运行Python文件
1. 确保已安装Python 3.6或更高版本
2. 双击 "水印工具.py" 或在命令行运行：python 水印工具.py
3. 程序会自动安装必要的依赖

方法2：使用批处理文件（仅Windows）
1. 双击 "运行水印工具.bat"
2. 程序会自动检查Python和依赖

📋 功能说明：

1. 图片导入
   - 点击"选择图片"导入单张或多张图片
   - 点击"选择文件夹"导入整个文件夹的图片
   - 支持格式：JPEG, PNG, BMP, TIFF

2. 水印设置
   - 水印文本：输入要添加的文字
   - 字体大小：调整文字大小（10-100）
   - 透明度：调整文字透明度（0-255）
   - 选择颜色：点击按钮选择文字颜色

3. 位置调整
   - 使用九宫格按钮快速定位
   - 在预览窗口中点击/拖拽调整位置

4. 导出图片
   - 导出当前图片：保存当前预览的图片
   - 批量导出：导出所有导入的图片

💡 使用技巧：
- 在预览窗口中直接点击可以调整水印位置
- 拖拽鼠标可以连续调整位置
- 支持实时预览，所有调整立即显示效果
- 导出时会自动添加"_watermarked"后缀

🔧 系统要求：
- Windows 7/8/10/11
- macOS 10.12+
- Linux（大多数发行版）
- Python 3.6+（如果从源码运行）

❓ 常见问题：

Q: 程序无法启动？
A: 请确保已安装Python 3.6或更高版本

Q: 图片无法加载？
A: 请检查图片格式是否支持（JPEG, PNG, BMP, TIFF）

Q: 水印位置不准确？
A: 可以先用九宫格按钮定位，再在预览窗口中微调

Q: 导出失败？
A: 请确保有足够的磁盘空间和写入权限

📞 技术支持：
如有问题，请检查：
1. Python版本是否符合要求
2. 图片格式是否支持
3. 是否有足够的磁盘空间
4. 是否有写入权限

🎉 祝您使用愉快！
