水印工具项目
============

📁 项目文件：
- 水印工具.py          # 主程序文件
- 启动水印工具.bat     # Windows启动器（推荐使用）
- 使用说明.txt         # 详细使用说明
- requirements.txt     # Python依赖列表

🚀 使用方法：
1. 双击 "启动水印工具.bat" 启动应用
2. 或者双击 "水印工具.py" 直接运行

✨ 功能特点：
- 图片导入（单张、多张、文件夹）
- 文本水印（自定义文本、大小、颜色、透明度）
- 位置调整（九宫格预设 + 鼠标拖拽）
- 实时预览
- 导出功能（单张 + 批量）

💻 系统要求：
- Python 3.6或更高版本
- Windows 7+ / macOS 10.12+ / Linux

🎯 助教测试：
1. 双击 "启动水印工具.bat"
2. 测试所有功能模块
3. 检查导出功能

项目已完成，可以直接使用！
